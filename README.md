# raindrop

A minimal Node CLI scaffold for the Raindrop.io CLI project.

## Requirements

- Node.js `>= 22.18.0`
- `pnpm`

## Install

```bash
pnpm install
```

## Build

```bash
pnpm build
```

## Run the example command

```bash
node dist/index.mjs example
node dist/index.mjs example --json
```

## Test

```bash
pnpm test
```
